# springboot-db

**说明**

数据库相关技术预研：

- 常用SQL数据库/NoSQL库/图数据库
- 数据库SQL版本管理
- 数据库连接池
- 数据库分库分表实现
- 数据库连接密码加解密
- 数据库多数据源实现
- 数据库多数据源+分布式事务管理实现
- ... ...
