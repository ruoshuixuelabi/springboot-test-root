## easyexcel、jxls、poi 对比 

- easyexcel 简单好用、效率高、社区活跃
- jxls 模板，稍微复杂，但可以应对更多场景，例如：模版填充数据、合并单元格、设置样式、公式计算等
- poi 复杂，但功能强大，例如：图片导出、公式计算等
