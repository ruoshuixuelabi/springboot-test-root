## 选型对比

* Apache 的 PDFBox，开源免费，功能相对较弱。
* Adobe 的 iText, 功能强，个人免费使用，AGPL 商用收费。
* iText5 资料网上相对较多，如果出现问题容易找到解决方案。
* PDFBox 和 iText8 的网上资料相对较少，如果出现问题不易找到相关解决方案。
